# 📊 Customer Churn Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing customer churn data to identify patterns and factors affecting customer retention. The dashboard helps in understanding why customers leave a service.

## 🛠️ Tools Used
- Power BI

## 📁 Dataset
- Telco Customer Churn Dataset (7000+ records)

## 📊 Features
- KPI cards showing total customers, churned customers, and average monthly charges
- Interactive filters (slicers) for gender, contract type, and payment method
- Visualizations to analyze churn distribution and behavior

## 📈 Key Insights
- Customers with month-to-month contracts have higher churn
- Electronic check users show higher churn rate
- Long-term contracts help in retaining customers

## 🎯 Conclusion
This dashboard helps businesses identify high-risk customers and take actions to improve customer retention.
